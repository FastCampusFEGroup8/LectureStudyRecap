## 폴더 및 파일 구조 규칙

### 폴더 구성

각 폴더는 특정 학습 기간을 나타내며, 시작 날짜와 종료 날짜를 포함합니다.

- 폴더명 형식: `YYYYMMDD-YYYYMMDD`

### 파일 구성

각 폴더 내에는 공부한 주제의 제목을 파일명으로 하는 마크다운 파일이 포함됩니다.

- 파일명 형식: `주제 제목.md`

### 마크다운 파일 내용

마크다운 파일에는 해당 주제에 대한 학습 내용 요약을 기록합니다

- **핵심 개념**: 주요 학습 내용 요약 📝
- **중요한 코드 예제**: 실제 적용한 코드 예제 💻
- **추가 참고 자료**: 학습을 돕는 추가 자료 링크 🔗
- **기타**: 기타 자료 📚
